Grid

The grid is a responsive percentage based grid that uses the same markup and class names as the bootstrap 2.0 grid. This is intentional to support oscar projects if required. The grid is generated in the file grid.less and contains several variables to easily reconfigure the grid if required, e.g. to have a different number of columns.

It collapses to a single column below a certain screen width. The first element in a row automatically has no left margin.

Below should be a comprehensive test of every grid element. The layout here is based on the default configuration of a 12 column grid. Details of classes and markup are included in the styleguide along with simpler examples.
